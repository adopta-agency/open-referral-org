Open Referral (Working)

This is my working area for evolving the Open Referral Human Services Data Specification (HSDS). HSDS is an open data standard for human services, and I'm helping define the next stage in its evolution as an API definition, as well as data schema. This is my (Kin Lane) dashboard for all the projects I'm touching as part of my Open Referral / HSDS work with Greg.
